# 48-200V-boost-conversion-at-a-duty-ratio-of-0.61-using-switched-LDL-configuration
Boost Converter (48 V–200 V) using switched Inductor-diode-inductor(LDL) configuration 

## Project Overview

This project involves the design, simulation, and hardware implementation of a high step-up DC-DC boost converter capable of converting a 48 V DC input into a regulated 200 V DC output at a duty ratio of 0.6. The project was developed to strengthen practical knowledge of power electronics, converter design, and hardware validation.

## Objectives

* Design a high step-up DC-DC boost converter.
* Validate theoretical analysis through MATLAB/Simulink simulations.
* Develop and test a hardware prototype.
* Compare simulation and experimental results to evaluate converter performance.

## Features

* Input Voltage: 48 V DC
* Output Voltage: 200 V DC
* Duty Ratio: 0.6
* MATLAB/Simulink modelling and simulation
* Hardware prototype implementation
* Output voltage and inductor current analysis
* Performance validation through experimental testing

## Tools and Technologies

* MATLAB
* Simulink
* Power electronic components (MOSFET, diode, inductor, capacitor)
* Oscilloscope
* Digital Multimeter
* DC Power Supply

## Outcomes

The converter successfully achieved the desired voltage conversion from 48 V to 200 V. Simulation results closely matched the hardware implementation, validating the converter design and providing practical experience in power electronics, circuit analysis, hardware prototyping, and performance evaluation.
